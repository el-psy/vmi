# vmi

## 使用Vue重写mi.com的首页  
本意是练习提高Vue的熟练度。  
其中的一些链接直接粘贴或者用"#", 图片链接直接复制粘贴，没有使用ajax。  
一些功能没有实现，比如 鼠标放到垂直导航上右边会弹出子菜单，图片轮播，下面的几个模块鼠标放到右上角的选项上下面的内容会变化等。
重点练习了页面布局。
